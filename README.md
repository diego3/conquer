### Relatório de compras do governo do Brasil

## Dependências

 - java 8
 - maven 3.5 >
 - docker 19 >
 - docker-compose 1.26 >

## Executando com o Docker
 Basta executar:
 ```cmd
docker-compose build
```
e depois um:
```cmd
docker-compose up
```
O mysql deverá estar rodando na porta 9096 e o Spring boot
na 8182

Acesse a aplicação pelo endereço: 
http://localhost:8182
